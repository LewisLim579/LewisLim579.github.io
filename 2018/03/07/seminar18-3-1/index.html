18년 3월 세미나 공유

Mar 07, 2018 in 세미나/컨퍼런스

2018년 3월 1차 ICT세미나/컨퍼런스

2018 MWC 리뷰 & 인사이트 콘서트

  • 내용 : 현장 기술 흐름을 파악하고 국내 모바일 분야 종사자에게 느낀 점과 기술 동향, 특이 사항 등을 강연
  • 장소 : 삼성동 코엑스 307호 (3F)
  • 일정 : 3월 8일 (목) 13:00 ~ 17:00
  • 주최 : 전자신문
  • 비용 : 사전등록 88,000원, 현장등록 110,000원
  • 신청 : 바로가기

Blockchain Meetup

  • 내용 : 컨센서스 알고리즘 & 토큰이코노미와 이더리움 기반의 크립토 게임개발 사례 소개
  • 장소 : 판교 The Planet 1층 Supex Hall
  • 일정 : 3월 8일 (목) 19시 00분 ~ 21시 00분
  • 주최 : SK텔레콤
  • 비용 : 무료
  • 신청 : 바로가기

빅데이터 입문자를 위한 파이썬 왕초보 과정

  • 내용 : 빅데이터 입문자에게 필요한 파이썬 문법과 빅데이터 기본 개념 강의
  • 장소 : 서울특별시 서초구 사임당로 173 서전빌딩 9층
  • 일정 : 2월 24일 (토) 13시 00분 ~ 18시 00분
  • 주최 : 비팬
  • 비용 : 무료
  • 신청 : 바로가기

제 2회 AI/블록체인 융합 IOT 비즈니스 개발 컨퍼런스 2018

  • 내용 : 인공지능 및 블록체인의 융합에 초점을 맞춰 기술의 융합을 통한 제품, 서비스, 비즈니스 모델 개발 소개
  • 장소 : 코엑스 E홀
  • 일정 : 3월 13일 (화) 10시 00분 ~ 17시 10분
  • 주최 : IoT DevCon 2018 조직위원회 / ㈜첨단
  • 비용 : 무료
  • 신청 : 바로가기

Digital Transformation Summit

  • 내용 : 디지털 트랜스포메이션의 현황과 영향을 분석하고, 기업들의 새로운 활로에 대한 방안을 고객 사례와 함께 제시
  • 장소 : 서울 송파구 올림픽대로 240 잠실 롯데호텔월드
  • 일정 : 3월 20일(화) 9시 00분 ~ 17시 10분
  • 주최 : Microsoft
  • 비용 : 무료
  • 신청 : 바로가기

The 6th (지급 결제의 혁신) All Payment Conference

  • 내용 : 지급결제 및 커머스 시장 내 최신 이슈를 공유, 변화하는 ICT 환경에 부합하는 새로운 지급 결제 서비스 및 커머스 플랫폼 등장에 대한 의견 공유
  • 장소 : 삼성동 코엑스 3층 Hall E
  • 일정 : 3월 22일 (목) 9시 30분 ~16시 40분
  • 주최 : 전자신문
  • 비용 : 사전등록 165,000원
  • 신청 : 바로가기
  • 후기 : All Payment Conference2013 후기

CDC World 2018

  • 내용 : 기업의 실제적인 클라우드, 데이터센터, 네트워크, 보안 전략 뿐 아니라, 미래를 준비하기 위한 인공지능과 머신러닝 등 클라우드 기반의 신기술 정보를 함께 논의
  • 장소 : 쉐라톤 서울 디큐브시티 호텔, 그랜드볼룸 (6F)
  • 일정 : 3월 22일 (목)
  • 주최 : 한국IDG㈜
  • 비용 : 선착순 120,000 단체등록 160,000 일반등록 190,000
  • 신청 : 바로가기
